CO2 Savings Help
This calculator lets the user determine carbon dioxide (CO2) emissions for given combustion fuel, biofuel, refrigerant charge, purchased gases, fuel use for transportation, and purchased electricity from the grid. Select the emission source type to select the emission from combustion fuel, biofuel, refrigerant charge, purchased gases, purchased electricity from the grid. Next select the emission source and corresponding fuel type. The emission output rate can be edited but will return to the default value when the electricity/fuel options are changed. For calculating emission from purchased electricity from the grid select Electricity under Emission Source Type. Location-based emissions use the grid-average emission factor (EPA e-GRID factor). The market based emission factors are calculated using residual mix from Green-e. Energy Type
The type of energy creating the emission.
Energy Source
Type of fuel creating the emission (used to set Fuel Type dropdown options). Can choose from Steam & Hot Water, Natural Gas, Petroleum-based fuels (including diesel and gasoline), biomass fuels (including biodiesel and landfill gas), coal, and other fuels (including coke).
Fuel Type
Specific fuel creating the emission (set by Energy Source).
Total Emission Output Rate
The total emissions per unit of energy use. By default, this is set by the dropdown options, but is changeable. The default values are only kg of carbon dioxide (not carbon dioxide equivalent).
Fuel Use
Amount of fuel used for creating the emission.
Electricity Use
Amount of electricity used creating the emission.
eGrid Sub-Region
eGrid sub-region where electricity is generated.
